# CareNest - Compassionate Care for Your Loved Ones

**CareNest** is a modern, responsive home-care service platform designed to connect families with professional caregivers. Whether it's for newborns, the elderly, or patients recovering from illness, CareNest provides a seamless way to book trusted professionals at your doorstep. This is a full-stack application built with **Next.js** and **MongoDB**.
